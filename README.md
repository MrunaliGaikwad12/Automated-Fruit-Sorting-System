# Automated-Fruit-Sorting-System

📌 𝐏𝐫𝐨𝐣𝐞𝐜𝐭 𝐎𝐯𝐞𝐫𝐯𝐢𝐞𝐰

🍎 Automated Fruit Sorting System Using ESP32

  - This project presents an ESP32-based automated fruit sorting system designed to classify fruits based on their color. The system detects the presence of a fruit using an IR sensor, identifies its color through a color sensor, and automatically sorts it into the appropriate category using a servo motor.
    
- A 16×2 LCD display provides real-time information about the detected fruit and sorting status. The project demonstrates embedded automation for agricultural and food processing industries by reducing manual effort and improving sorting efficiency.

  ---

 ❓ 𝐏𝐫𝐨𝐛𝐥𝐞𝐦 𝐒𝐭𝐚𝐭𝐞𝐦𝐞𝐧𝐭

Manual fruit sorting is time-consuming, labor-intensive, and prone to human error. Agricultural and food processing industries require an efficient, accurate, and low-cost automated solution for sorting fruits based on their characteristics. This project addresses these challenges by implementing a smart color-based fruit sorting system.

The system consists of:

**IR Sensor → ESP32 Processing Unit → Color Detection → Servo Motor → LCD Display**

- IR Sensor:detects the presence of a fruit.
- Color Sensor : identifies the fruit color.
- ESP32: processes sensor data and determines the fruit category.
- Servo: Motor rotates to direct the fruit into the appropriate collection bin.
- LCD: displays the detected fruit color and sorting status.

1. The IR sensor detects the arrival of a fruit.
2. The ESP32 activates the color sensor to identify the fruit's color.
3. The detected color is compared with predefined color values.
4. The ESP32 determines the appropriate sorting category.
5. The servo motor rotates to direct the fruit into the corresponding collection bin.
6. The LCD displays the detected color and sorting status.
7. The servo returns to its initial position and waits for the next fruit.
